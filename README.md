With_Graph - полноценный проект с визуализацией (и почти без костылей), сделанный в WF.
With_Graph.rar - все исходники проекта
With_Graph.exe - исполняемый файл
Интерфейс:
- два поля для ввода данных, данные можна вводить с клавиатуры или загрузить из файла
- поле для ответа
- кнопка "File" - для загрузки данных для обработки из файла (в файле должны храниться две строки, длина каждой из которых не превосходит 1000 символов)
- переключатель "with graph" - если включен - обработка произойдет с визуализацией, нет - просто выдаст ответ
- кнопка "Count up!" - запускает обработку введенных данных
Ведется лог выполненных действий, хранящийся в файле log.txt
Остальные поля заолняются программой в процессе обработки, и отображают ее различные этапы.
Если запущен режим визуализации - обработка переходит на следующий этап при нажатии клавиши "BackSpace"


Without_Graph - упрощенная консольная программа без визуализации (и совсем без костылей) которая реализует непосредственно алгоритмическую составляющую - и позволяет проверить корректность работы алгоритма/провести скоростные тесты/попытаться "взломать" реализацию и просто разобраться в самом алгоритме не отвлекаясь на визуализацию. Считывает данные из "input.txt", выводит в "output.txt" - все стандартно. Формат ввода: две строки, длина каждой из которых не превосходит 1000 символов
Without_Graph.cpp - исходный код программы
Without_Graph.exe - исполняемый файл
